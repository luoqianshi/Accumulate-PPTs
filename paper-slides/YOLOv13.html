Background & Motivation

研究背景与动机

现有方法的局限

  • 卷积架构（YOLO11及之前）：仅能在固定感受野内进行局部信息聚合
  • 自注意力机制（YOLOv12）：受限于计算成本，只能进行局部区域计算
  • 自注意力本质上是成对像素相关性建模，无法捕获多对多的高阶关联

核心挑战

  • 现有YOLO模型缺乏全局高阶语义关联建模能力
  • 传统超图方法使用手工设定阈值构建超边，鲁棒性不足
  • 复杂场景下检测性能存在瓶颈
目标：突破成对关联建模限制，实现全局高阶关联感知
Core Contributions

核心贡献

🔬 HyperACE 机制

基于自适应超图计算，自适应探索潜在高阶关联，实现跨位置、跨尺度特征融合与增强

🔄 FullPAD 范式

全管线聚合-分发范式，将关联增强特征分发到Backbone、Neck、Head全流程，实现细粒度信息流

⚡ DS 系列轻量化模块

基于深度可分离卷积设计轻量化模块，参数量减少30%，GFLOPs减少28%，精度几乎无损

YOLOv13-N 相比 YOLOv12-N 提升 AP50:95 1.5% | 相比 YOLO11-N 提升 3.0%

HyperACE：超图自适应关联增强

双分支并行设计

🌐 高阶感知分支（C3AH）

  • 基于自适应超图计算建模全局高阶关联
  • K个并行C3AH模块探索不同潜在关联

自适应超图计算流程

1

自适应超边生成

全局池化 → 可学习原型 → 多头相似度 → 连续参与矩阵 A ∈ [0,1]N×M

2

超图卷积

超边聚合（V→E）→ 线性投影 → 超边分发（E→V）→ 特征更新

3

特征融合

高阶 + 低阶 + 捷径三分支拼接 → 1×1卷积融合输出

Technical Detail

自适应超图计算：关键创新

传统超图 vs 自适应超图

  • 传统方法：手工阈值判定相关性，使用二值关联矩阵 H ∈ {0,1}N×M
  • 本文方法：自适应学习参与度，使用连续参与矩阵 A ∈ [0,1]N×M
每个超边连接所有顶点，每个顶点以连续可微的贡献度参与

核心公式

  • 全局原型：P = P₀ + ϕ(fctx)，动态偏移 + 可学习原型
  • 多头相似度：s̄i,m = 平均(⟨ẑiτ, p̂mτ⟩ / √dh)
  • 参与矩阵：Ai,m = exp(s̄i,m) / Σj exp(s̄j,m)
  • 超图卷积：fm = σ(We Σi Ai,m xi)，xi = σ(Wv Σm Ai,m fm)
Lightweight Design

DS 系列轻量化模块

DSConv

深度可分离卷积 + BN + SiLU，作为基础构建单元

DS-Bottleneck

3×3 DSConv + k×k 大核 DSConv 级联，可选残差连接

DS-C3k

CSP-C3 结构 + DS-Bottleneck，用于HyperACE低阶分支

DS-C3k2

C3k2 结构 + DS-C3k 模块，Backbone/Neck 基础模块

参数量减少高达 30% | GFLOPs 减少高达 28% | 精度损失仅 0.1% AP50

Visualization

自适应超边可视化

超边 1：前景物体间关联

建模多个前景目标之间的高阶交互，如滑雪者与滑雪杖之间的关联

超边 2：场景内元素关联

捕获场景内多个元素的关联，如多辆车与多个交通灯之间的协同关系

超边 3：前景-背景关联

建模前景物体与背景场景之间的关联，如网球拍与网球场、棒球手套与棒球场的关联

HyperACE 能够自适应学习场景中潜在的多对多高阶视觉关联，而非仅依赖低阶成对关联
Conclusion

总结与展望

核心总结

  • 提出 HyperACE：首个用于目标检测的自适应超图关联增强机制
  • 提出 FullPAD：全管线聚合-分发范式，实现细粒度信息流
  • 提出 DS 系列模块：深度可分离卷积轻量化设计
  • 在 MS COCO 上全面超越所有先前YOLO版本
  • 轻量化变体（N/S）优势最显著，符合YOLO系列"更准、更快、更轻"目标

关键数据

  • YOLOv13-N: AP50:95 41.6%，参数仅 2.5M
  • YOLOv13-S: AP50:95 48.0%，延迟 <3ms (T4)
  • YOLOv13-X: AP50:95 54.8%，SOTA 性能
